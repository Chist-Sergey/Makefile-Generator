# Usage  

## Step 1.  

In Terminal/Console, go to the place where  
the genmakefile script is located.  

## Step 2.  

Run this command in your Terminal/Console:  

    python genmakefile.py  

Use "python3" instead on macOS/Linux.  

## Step 3 (optional).  

See "[extra.md](./extra.md)".  

# That's it.
